# Dice Roll Simulator
A simple Python program that simulates rolling two six-sided dice. The program repeatedly asks the user whether they want to roll the dice and displays two randomly generated values until the user chooses to exit.

## Features
- Simulates rolling two dice.
- Generates random numbers between 1 and 6.
- Allows multiple rolls without restarting the program.
- Handles invalid user input.

## Technologies Used
- Python 3
- `random` module
